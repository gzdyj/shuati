# 单选题刷题系统开发计划

## 项目概述
- **项目名称**: QuizMaster - 单选题刷题系统
- **项目类型**: 前后端分离的Web应用
- **核心技术栈**: 前端Vue 3 + Vite，后端Spring Boot (Java)
- **目标用户**: 需要进行单选题练习和备考的用户
